Hands-on Lab Part 2: Creating Watson Discovery service (15 min)

Objective:

  • How to create a Watson Discovery service.

To access the resources and services that the IBM Cloud provides, you need an IBM Cloud account.

If you already have an IBM Cloud account, you can directly move on to creating Watson Disovery Service. Otherwise, please follow the previous lab to create an IBM Cloud account.

Create a Watson Discovery instance

Step 1 - Login to your IBM Cloud account using your registered email id and password.

Step 2 - Click on the Create resource button on your dashboard.

Step 3 - Search for Discovery in the search field or find it in the AI category of the catalog and then click on that tile.

Step 4 - A page will offer you to create the service. Select London as the region. Change the name of the service or keep the default, if you wish. Verify that the Lite plan is selected, and then click Create.

Note: If you created a Watson Discovery instance in the past using the Lite plan, you won't be allowed to create a second instance. You can use the instance you already created for this exercise or you'll need to delete your previous instance and create a new one. Also, there is no charge associated in creating a Lite plan instance of the Watson Discovery Service.

You will be redirected to the Resource List page where you can see the Discovery Service you just created as active.

You're all set for now.
